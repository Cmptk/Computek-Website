How to Publish This Website for Free Using GitHub Pages

  1. Go to github.com and create a free account (if you don’t already have one).
  2. Click the + (plus) icon in the top right, then select "New repository".
  3. Name the repository something like computek-website, and click "Create repository".
  4. On your computer, make a folder and place your index.html file and logo.png inside it.
  5. Go back to GitHub and upload your files to the new repository using the "Add file" → "Upload files" button.
  6. After uploading, go to the repository’s Settings → Pages tab (in the sidebar).
  7. Under "Source", choose main branch and folder /root or /docs and click "Save".
  8. GitHub will give you a free website address like:
    https://yourusername.github.io/computek-website
